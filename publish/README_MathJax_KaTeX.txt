사용법(가장 확실):
1) 파일을 더블클릭해서 브라우저로 열면 수식이 바로 렌더링됩니다.
2) 이 HTML을 '스크립트/스타일 삽입이 허용되는' 웹페이지(티스토리, GitHub Pages 등)에 올리면 그대로 렌더링됩니다.

주의:
- 네이버 블로그는 보안 정책상 <script> 같은 태그가 글 본문에서 제거/차단되는 경우가 많아
